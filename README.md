# Vue_projects
（新闻移动客户端+后台管理系统）+前后端分离+Vue+Element+ThinkJS

News_Nuxt_SPA 移动端

News_Nuxt_SSR 后台管理系统

New_ThinkJS 后端

News_Nuxt_SPA、News_Nuxt_SSR、New_ThinkJS 下载依赖: npm install

News_Nuxt_SPA、News_Nuxt_SSR 运行：npm run dev

New_ThinkJS 运行：npm start
